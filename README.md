# Voucher Blog API

This repo contains all necessary API for Voucher Blog applications to run.



## Instructions

Note: remember to add your `.env ` before running

* Without Docker

```bash
  npm install
  npm start
```

* With Docker

```bash
  docker build -t voucher-blog-api:main .
  docker run -d -p 3000:3000 voucher-blog-api:main
```
